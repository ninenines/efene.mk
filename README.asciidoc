= Efene.mk

http://efene.org/[Efene] plugin for http://erlang.mk/[Erlang.mk]

[source,make]
----
BUILD_DEPS = efene

BUILD_DEPS = efene.mk
dep_efene.mk = git https://github.com/ninenines/efene.mk master
DEP_PLUGINS = efene.mk

include erlang.mk
----

The command `make` will compile all Efene files in the 'src/'
directory.
